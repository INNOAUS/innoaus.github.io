Philips Hue iOS SDK 사용하기

2018-04-25
Innoaus R & D Center

Philips Hue iOS SDK를 Xcode 프로젝트에 추가하기

Prerequisites

Downloads

  • HueSDK from Philips Github: HueSDK.framework
    • 해당 git를 ‘download or clone’하면 HueSDK를 얻을 수 있다.
  • HueSDK iOS Framework Guide: HueSDK iOS Framework Guide

env.

  • Philips Hue Ambiance Starter Kit
    • Hue Bridge
    • Hue 조명
  • macOS 10.13 (High Sierra)
    • Xcode 9
    • iOS 11.3
    • Swift4

Getting Started

1. 새 Xcode 프로젝트 만들기

  • Project Name: HueController
  • Template: Single View
  • Language: Swift

2. Framework 추가하기

  1. Project Directory에 Dependencies 폴더 생성하고, HueSDK.framework 복사

  • ~/home/HueController/Dependencies/HueSDK.framework (for device)

  • 주의: HueSDK iOS Framework Guide를 보면 device, simulator의 Framework를 한번에 추가하는 방법으로 알려주는데, 빠른 실행과 결과를 보기 위해 device Framework만 추가한다.

  1. Xcode에서 Dependencies그룹을 생성하고, 위에서 복사한 HueSDK.framework을 drag&drop으로 Add Reference

  2. Target > Build Phases > Link Binary With Libraries에 Project Navigation에서 HueSDK.framework을 drag&drop으로 추가

  3. Target > General > Embedded Binaries에 HueSDK.framework 추가

  4. Objective-C Briding Header추가를 위해 Header.h 추가

  • New File > Header file

    • File Name: Header.h

  • Target > Build Settings > Objective-C Bridging Header 수정

  • Header.h에 다음 내용 추가

3. Enable Bitcode

  • No

4. Build & Run

  • HueSDK.framework for device만 추가했기때문에,
    빌드 전에 빌드대상을 Device 또는 Generic iOS Device를 선택한다.

Hue SDK Guide

https://www.developers.meethue.com/ 에 회원등록하면, 자세한 Hue SDK Guide를 확인할 수 있다.
